### 📈 O Projeto: Worky

O **Worky** utiliza técnicas de Web Scraping e Processamento de Linguagem Natural (NLP) para analisar grandes volumes de vagas.
- **Análise Semântica:** Identificação de habilidades reais vs. termos genéricos.
- **Tendências:** Monitoramento de salários e tecnologias em alta.
- **Visualização de Dados:** Gráficos simplificados para tomada de decisão rápida.
